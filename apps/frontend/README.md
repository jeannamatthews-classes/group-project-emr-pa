# Frontend Spin-Up

Use the root `README.md` for full project setup.

## Quick Start

From the project root:

```bash
npm run dev:frontend
```

Frontend URL:

```text
http://localhost:5173
```

## Before You Start

- Make sure `npm install` has already been run from the root folder.
- For normal app use, the backend should also be running.

## Build

From the project root:

```bash
npm run build -w apps/frontend
```
